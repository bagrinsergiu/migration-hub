## Требуется выполнить вручную

⚠️ **Установка пакета через Composer**

Из-за проблем с окружением (PHP 7.2 вместо требуемого 7.4+), автоматическая установка не удалась. 

**Необходимо выполнить вручную:**

1. Убедитесь, что используется PHP 7.4 или выше:
   ```bash
   php -v  # Должно быть >= 7.4
   ```

2. Если нужно переключить версию PHP, используйте:
   ```bash
   # Для Ubuntu/Debian с ondrej/php PPA:
   sudo update-alternatives --set php /usr/bin/php7.4
   # или
   sudo update-alternatives --set php /usr/bin/php8.0
   ```

3. Установите пакет:
   ```bash
   composer require google/apiclient
   ```
   
   Или если зависимость уже в composer.json:
   ```bash
   composer update google/apiclient
   ```

4. Проверьте установку:
   ```bash
   composer show google/apiclient
   ```

5. Проверьте, что классы доступны:
   ```bash
   php -r "require 'vendor/autoload.php'; var_dump(class_exists('Google_Client'));"
   ```
