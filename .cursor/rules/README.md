# Правила Cursor для проекта

Этот каталог содержит правила для AI-агента в Cursor.

## Файлы правил

- `mcp-usage.mdc` - Правила эффективного использования всех MCP серверов
- `serena.mdc` - Правила использования Serena (если существует)

## Как работают правила

Cursor автоматически загружает файлы `.mdc` из `.cursor/rules/` и применяет их к AI-ассистенту.

Правила помогают агенту:
- Выбирать правильные инструменты для задач
- Использовать MCP серверы эффективно
- Следовать лучшим практикам проекта

## Обновление правил

Просто отредактируйте файлы `.mdc` в этом каталоге. Изменения вступят в силу после перезапуска Cursor.
