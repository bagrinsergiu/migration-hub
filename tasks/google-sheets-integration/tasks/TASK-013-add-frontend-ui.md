# TASK-013: Добавление UI для управления Google Sheets

## Статус: ⏳ В ожидании

**Зависимости:** TASK-009

## Описание

Добавить пользовательский интерфейс в дашборд для управления подключениями Google Sheets. UI должен позволять подключать таблицы, просматривать список подключенных таблиц, синхронизировать данные и привязывать листы к волнам.

## Что нужно сделать

### 1. Создать компоненты React

**Директория:** `dashboard/frontend/src/components/GoogleSheets/`

**Компоненты:**

#### `GoogleSheetsList.tsx`
Список подключенных таблиц:
- Отображение таблиц с информацией (название, последняя синхронизация, привязанные волны)
- Кнопки: синхронизировать, удалить, просмотреть детали

#### `GoogleSheetsConnect.tsx`
Форма подключения новой таблицы:
- Поле для ввода spreadsheet_id
- Кнопка "Подключить"
- OAuth авторизация (если нужно)

#### `GoogleSheetsLinkWave.tsx`
Привязка листа к волне:
- Выбор таблицы и листа
- Выбор волны из списка
- Кнопка "Привязать"

#### `GoogleSheetsSyncStatus.tsx`
Статус синхронизации:
- Индикатор последней синхронизации
- Кнопка "Синхронизировать сейчас"
- Прогресс синхронизации (если идет)

### 2. Добавить страницу в роутинг

**Файл:** `dashboard/frontend/src/App.tsx` или роутер

Добавить маршрут: `/google-sheets`

### 3. Интегрировать с API

Создать API клиент методы:
- `connectGoogleSheet()`
- `getGoogleSheetsList()`
- `syncGoogleSheet()`
- `linkSheetToWave()`

### 4. Добавить навигацию

Добавить пункт меню "Google Sheets" в навигацию дашборда.
